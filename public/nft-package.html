Préparation du NFT interactif pour IPFS

Votre NFT interactif comprend plusieurs fichiers:

  • index.html - Le code HTML principal du NFT interactif
  • shacker01.jpg - L'image de fond
  • character.png - Le personnage interactif

Pour télécharger et utiliser ce package:

  1. Téléchargez chaque fichier individuellement avec les liens ci-dessous
  2. Créez un dossier sur votre ordinateur nommé "shacker-nft"
  3. Placez tous les fichiers téléchargés dans ce dossier
  4. Vous pouvez maintenant tester localement en ouvrant index.html dans votre navigateur
  5. Pour le déployer sur IPFS, vous aurez besoin d'uploader l'ensemble du dossier

Instructions pour uploader sur IPFS

Pour uploader ce package sur IPFS, vous avez plusieurs options:

Option 1: NFT.Storage (Recommandée)

  1. Allez sur https://nft.storage
  2. Créez un compte ou connectez-vous
  3. Cliquez sur "Upload"
  4. Sélectionnez le dossier entier "shacker-nft" contenant tous les fichiers
  5. Une fois l'upload terminé, vous recevrez un CID IPFS que vous pourrez utiliser pour votre NFT

Option 2: Pinata

  1. Allez sur https://pinata.cloud
  2. Créez un compte ou connectez-vous
  3. Cliquez sur "Upload" puis "Folder"
  4. Sélectionnez le dossier "shacker-nft" contenant tous les fichiers
  5. Vous recevrez un CID IPFS pour l'ensemble du dossier

Intégration dans les métadonnées NFT

Une fois que vous avez obtenu le CID IPFS, vous devez l'utiliser dans vos métadonnées NFT comme suit:

{
  "name": "S.H.A.C.K.E.R. #01",
  "description": "Une créature démoniaque aux yeux jaunes flamboyants et aux petites cornes. NFT rare de la collection Shackers OG sur Ethereum. NFT interactif avec mini-jeu intégré.",
  "image": "ipfs://VOTRE_CID_ICI/shacker01.jpg",
  "animation_url": "ipfs://VOTRE_CID_ICI/index.html",
  "attributes": [
    {
      "trait_type": "Gender",
      "value": "Male"
    }
  ]
}

Notes importantes:

  • Remplacez "VOTRE_CID_ICI" par le CID que vous avez obtenu lors de l'upload
  • Le champ "animation_url" permet aux plateformes comme OpenSea d'afficher votre NFT interactif
  • Le champ "image" sera utilisé comme image par défaut
