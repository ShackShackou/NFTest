Upload Direct sur NFT.Storage

Cette fonctionnalité vous permet d'uploader directement vos fichiers sur IPFS via NFT.Storage - pour un stockage permanent et décentralisé.

Les fichiers uploadés sur IPFS sont accessibles via leur CID unique et peuvent être utilisés dans vos métadonnées NFT.

Utilisation dans les métadonnées NFT

Une fois vos fichiers uploadés, intégrez-les dans vos métadonnées NFT en utilisant les URLs IPFS générées:

{
  "name": "S.H.A.C.K.E.R. #01",
  "description": "Une créature démoniaque aux yeux jaunes flamboyants et aux petites cornes.",
  "image": "ipfs://bafkreiaylxpfsgvouqwwqopxvfrqe7qasdyunbgprqp2jkrz37gssstpry",
  "animation_url": "ipfs://bafkreidivzimqfqtoqxkrpge6bjyhlvxqs3rjv5yze7uus7unnbgpyzpce",
  "attributes": [
    {
      "trait_type": "Type",
      "value": "Demon"
    }
  ]
}

Note: L'URL animation_url permet à OpenSea et autres marketplaces d'afficher votre HTML interactif à la place de l'image statique.
